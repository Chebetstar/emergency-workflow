QuickAid: Emergency Triage Tracker

A simple system to register and track patients based on urgency level.

About This Project

QuickAid is a prototype Emergency Room Workflow system designed to help hospital staff track incoming patients based on urgency level. It allows patient registration, urgency classification, timestamp logging, and automatic color-coded triage queues. It also saves patient data locally in the browser, so nothing is lost on refresh.
