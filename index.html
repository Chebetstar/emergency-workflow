QuickAid: Emergency Triage Tracker

A simple system to register and track patients based on urgency level.

About This Project

QuickAid helps hospital staff track incoming patients by urgency level. It registers patients, logs timestamps, tracks status changes, and displays notifications. Data is saved in the browser.
